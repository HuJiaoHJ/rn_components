## React Native 组件库

### 运行项目

此项目是使用 `react-native init rn_components` 生成的，所以运行起来比较简单，即

```bash
// 安装依赖
yarn
// 运行 js bundle server
npm run start

react-native run-ios
// or
react-native run-android
```
